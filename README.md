# Python-project-Student-grade-management-system-
So this project is about student grade management system how data is managed. basically it is a mini digital system to manage the data like adding names ronn no. avg marks attendance.  
We can add student,update marks, search students ,display report cards.
FILENAME: "student.txt"
Firstly we have to login it using the assword.
And then options will comes: Add students, Search student,Update marks,Show toppers,Report cards,Exit.
TECHNOLOGIES USED: conditional statements , loops, pandas, oop(object oriented programming), file handling, exception handling,def,__init__

After login, the following options are available:
Add Student
Enter roll number, name, attendance, and marks
System calculates average and grade automatically
Search Student
Search by:
Roll number
Name
Update Marks
Update marks for existing student
Average and grade are recalculated
Show Toppers
Displays student(s) with highest average marks
Report Cards
Displays all students in table format using pandas
Exit
Saves data and exits program
